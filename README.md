## 目前的编译已修复卡刷无法开机的问题。

我的固件很挑卡。不过目前已解决了卡刷不开机的问题。

方法两种：

- 下载r4s固件刷到卡上，然后再刷r2s即可解决挑卡问题，就能开机

- 用软件卡写到一半就拔出，再写卡完了即可清除分区里的垃圾。（感谢Aney提供）
以上这两种方法即可解决挑卡不开机问题！
